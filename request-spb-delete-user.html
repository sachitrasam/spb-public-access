Please use in-app Contact Us feature if a particular user needs to delete the account or their data used in the app. In case the an account deletion: * The email address used for the creation of the account will be deleted from the system. * After a successful deletion, the same email can be used to create a new account. In case of data deletion: * All the data collected as user performance data will be deleted: In such a case, following features will not display any data: - All performance statistics and graphs - User centric quiz recommendations
